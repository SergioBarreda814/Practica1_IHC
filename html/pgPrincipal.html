Objetivo

Desarrollar dos pruebas de concepto de interfaces interactivas para la Práctica N°1 de Interacción Humano-Computadora, con el fin de explorar y comparar diferentes modalidades de interacción con computadoras, aplicando los conocimientos de paradigmas de interacción. Promoviendo el trabajo en equipo, se busca adquirir competencias en modelos de interacción derivados de estilos de interacción.

Paradigma de Interacción 1: Interfaz de Voz

Este paradigma utiliza la API de reconocimiento de voz para permitir al usuario controlar el objeto mediante comandos de voz. Cuando se detecta voz, se transcribe el texto reconocido y se compara con palabras clave como "arriba", "abajo", "izquierda" y "derecha". Si se reconoce alguna de estas palabras, se llama a la función moveObject(direction) con la dirección correspondiente como argumento, de manera similar a la interacción mediante teclado. Si la palabra no coincide con ninguna de las palabras clave esperadas, se muestra un mensaje en la consola indicando que la palabra no fue reconocida.
